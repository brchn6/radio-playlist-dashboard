## Integration Notes

### Format for dashboard stations.json

The dashboard expects this format:

```json
{
  "id": 1,
  "slug": "glglz",
  "name": "גלגלצ",
  "stream_url": "https://glzwizzlv.bynetcdn.com/glglz_mp3",
  "proxy_port": 87XX,
  "color": "#hexcolor",
  "enabled": 1,
  "created_at": "2026-07-13T12:46:45",
  "website": "https://glglz.co.il"
}
```
